## 二、决定的理由

## 1. 审查基础

专利权人在答复无效宣告请求书时提交了权利要求书修改替换页，其中在授权公告文本的基础上，将权利要求2的特征“该晶型II使用 $\mathrm{Cu - K}\alpha$ 辐射，X射线粉末衍射图谱如图1所示”补入权利要求3。

请求人不认可上述修改，认为在本专利实施例1-5没有逐一对得到的产物进行晶型的确认，说明书也没有指明图1对应于哪个实施例得到的晶体的情况下，权利要求1-3属于不同的技术方案，因此将其中的参数进行合并形成了新的技术方案，超出了原申请文件记载的范围，不符合专利法第33条的规定。

专利权人认为，本专利权利要求1-3、以及说明书实施例1-5涉及的是同一种晶体，即晶型II，权利要求1-3仅仅是用不同的方式对该晶体进行了限定；X射线衍射图谱对于晶体表征是指纹性的，不同实施例制备得到的相同晶体，X射线衍射图谱也是相同的。

经查，本专利权利要求1-3涉及7-乙基-10-羟基喜树碱的晶型Ⅱ，分别采用10个X射线粉末衍射峰、图1以及红外谱带进行了限定(参见案由部分)。

根据本专利说明书的记载，本发明涉及7-乙基-10羟基喜树碱的两种新晶型及其制备方法……7-乙基-10-羟基喜树碱的不同晶型溶解度和湿度稳定性不同而且差别很大，而溶解度是制备抗肿瘤药物过程中的重要指标，溶解度过小会严重影响药物的品质。中国专利文件CN101686968A公开了一种7-乙基-10-羟基喜树碱的多晶型物，其X一射线衍射图在 $10.9\pm0.2$ 、 $13.2\pm0.2$ 、 $23.9\pm0.2$ 和 $26.1\pm0.22\theta$ 度处有峰。但是，该7-乙基-10-羟基喜树碱的多晶型物的溶解度不高。针对现有技术的不足，本发明提供了两种7-乙基-10-羟基喜树碱新晶型，分别记为晶型Ⅱ和晶型Ⅲ（参见本专利说明书第0001、0005-0007段）。

本专利说明书第0009-0018段以及附图1-6提供了该晶型II的一些表征参数，包括使用 $\mathrm{Cu - K}$ $\alpha$ 辐射，以 $2\theta$ 角度表示的X射线粉末衍射位置、X射线粉末衍射图谱、晶体结构参数、热分析图谱、红外图谱、拉曼图谱等。实施例1-5提供了采用不同方法和条件获得晶型II的过程。

可见，本专利的发明目的之一即为在现有技术公开的7-乙基-10-羟基喜树碱的已知晶型的基础上，提供的溶解度更好的晶型Ⅱ；在制备获得晶型Ⅱ之后，本发明采用本领域的一些常规检测技术对该晶体进行了测定和表征。

合议组认为，首先，本领域技术人员公知，X射线粉末衍射分析技术是对众多微小颗粒样品的共同检测分析，其分析结果具有良好的晶型物质代表特性，无论是对于不同的化合物晶体，还是相同化合物的不同晶型，均具有特定的专属性，是本领域进行晶型检测分析的最主要的技术手段。

本案中，权利要求1-3的保护主题均已明确限定为晶型II；通过本专利说明书提供的以上信息亦可知，图1的X射线粉末衍射图谱、以 $2\theta$ 角度表示的X射线粉末衍射位置以及红外谱带均为所述晶型II所具有的基本特征，也就是说，将上述晶型II的信息组合后所限定的技术方案仍然是晶型II，没有超出原申请文件所记载的范围。在没有相反证据的情况下，仅因为说明书没有对每一个实施例得到的晶型分别进行测定和表征，以及没有指明图1对应于具体哪个实施例的产品，就质疑实施例1-5的产品“不一定是同一晶型，以及图1不能代表任何一个实施例”缺乏依据。因此，请求人主张权利要求3修改超范围，不符合专利法第33条的理由不能成立。

其次，从修改方式而言，权利要求3的所述修改属于权利要求的进一步限定，符合专利审查指南第四部分第三章第4.6.2节有关无效程序中修改方式的规定。

因此，本案所依据的文本是专利权人于2024年06月03日提交的权利要求书（共10项）和授权公告文本的其他部分。

## 2. 证据认定

证据1是中国专利文献，证据2是中文书籍节选，专利权人认可证据1和2的真实性。

经核实，合议组对上述证据的真实性予以认可，其公开日均早于本专利申请日，可以作为现有技术评价本专利权利要求的新颖性和创造性。

## 3. 关于新颖性和创造性

专利法第 22 条第 2 款规定，新颖性，是指该发明或者实用新型不属于现有技术；也没有任何单位或者个人就同样的发明或者实用新型在申请日以前向国务院专利行政部门提出过申请，并记载在申请日以后公布的专利申请文件或者公告的专利文件中。

专利法第22条第3款规定，创造性，是指与现有技术相比，该发明具有突出的实质性特点和显著的进步，该实用新型具有实质性特点和进步。

请求人认为，（1）证据1中公开了一种7-乙基-10-羟基喜树碱的多晶型物及其X射线粉末衍射谱图及数据，与本专利权利要求1限定的特征峰在衍射角度存在细微差别，但这属于常规误差，两者技术方案实质相同，属于相同技术领域，要解决的技术问题和产生的技术效果相同，权利要求1不符合专利法第22条第2款的规定。（2）证据1与权利要求2的主要区别特征在于相应衍射峰的强度(CPS)，尽管本专利说明书实施例1验证了晶型II具有较好的溶解度，但由于本专利说明书未对其图1所示XRD图谱的对应材料作说明，据此不能判断图1所示XRD图谱即是实施例1产品，权利要求2实际解决的问题是提供一种可替代的7-乙基-10-羟基喜树碱晶型，本领域技术人员通过常规的有限的实验手段即可对产品的晶型进行调整，得到溶解度等性能相近的相似晶型产品。因此，权利要求2的技术方案相对于证据1是显而易见的，没有突出的实质性特点和显著的进步，不符合专利法第22条第3款的规定。（3）证据1还公开了其多晶型物的红外图谱，与本专利权利要求3进一步限定的谱带波数仅存在细微差别，因此，该权利要求也不具备创造性。证据2用于证明证据1中的XRD测试步骤对应于本专利的 $\mathrm{Cu - K}\alpha$ 辐射。

专利权人认为，权利要求1-3的方案与证据1虽然都是7-乙基-10-羟基喜树碱晶型，但晶型结构存在明显差别，XRD图谱显示衍射峰数量、位置和强度均不同，因此技术方案实质不同，要解决的技术问题和产生的技术效果也不同，因此具备新颖性和创造性。

对于有机化合物晶体发明而言，一般情况下，若权利要求未指明晶型且仅采用部分特征峰限定的形式，表明专利权人意欲获得一个相对较宽的保护范围，而不是局限于实施例实际制备得到的具体晶型产品，即与使用X-射线粉末衍射全图限定的权利要求的保护范围是不同的。但本案中，如前所述，在权利要求1-3已经明确限定其保护主题均为晶型II的前提下，权利要求1-3的技术方案应当理解为采用不同的参数特征对同一晶型进行表征。

进而，通过上表对照可见，本专利的晶型II与证据1公开的晶型在X射线粉末衍射特征峰、X射线粉末衍射图谱、红外谱带以及红外图谱均存在较为明显的差异，例如，本专利晶型II在10.80处为最强峰，而证据1在该位置附近则具有10.9的次强峰和10.4的峰；本专利晶型在24.52和26.84处的强峰与证据1在23.9和26.1的两个强峰无论在位置抑或强度上也存在较大差异，且所述差异超出了本领域技术人员能够认为的常规误差的范畴。因此，证据1晶型在衍射峰数量、峰位置、峰强度等方面与本专利晶型II存在区别，不足以认定二者属于相同晶型，请求人主张的权利要求1不具备新颖性，不符合专利法第22条第2款规定的理由不能成立。

关于本专利晶型II的技术效果，如前所述，本专利的发明目的在于获得溶解度更高的7—乙基—10—羟基喜树碱晶型，说明书背景技术部分所提及的专利文献即为证据1。

根据说明书的记载，本发明在实验例中测定了“实施例1得到的7-乙基-10-羟基喜树碱晶型Ⅱ、实施例6得到的7-乙基-10-羟基喜树碱晶型Ⅲ以及中国专利文件CN101686968A（注，即证据1）报道的晶型(记为晶型I)，在 $25^{\circ}\mathrm{C}$ 下 $\mathrm{pH} = 1.2$ 水溶液中的溶解度”。结果显示，“晶型Ⅱ、Ⅲ的溶解度比晶型I的溶解度大，晶型I、Ⅱ、Ⅲ的最大溶解度是0.46、0.98和 $1.27\mathrm{mg / L}$ 。但是晶型Ⅱ和Ⅲ的溶解度不断下降，2h-4h溶解度平衡，溶解度达平衡后，晶型I、Ⅱ、Ⅲ的溶解度分别是0.45、0.65和 $0.80\mathrm{mg / L}$ ，可知，本发明提供的晶型Ⅱ、Ⅲ的溶解度得到了很大的提高”（参见本专利说明书第0078-0081段）。可见，较之证据1的晶型I，本发明取得了溶解度提高的技术效果。

请求人主张本专利权利要求2-3不具备创造性的依据仅为“本专利说明书中未对图1所示XRD图谱的对应材料(实施例)作说明，据此不能判断图1所示XRD图谱即是实施例1产品的XRD图谱，不能认可权利要求2-3的产品能够解决溶解度的技术问题”，在前文1.审查基础部分已经对该观点进行了针对性回应。

因此，在专利权人已经证明关于所保护的晶体能够区别于现有技术并具有有益技术效果，完成了初步举证责任的情况下，请求人主张的权利要求2-3保护的晶型II不具备创造性，不符合专利法第22条第3款规定的理由亦不能成立。

## 4. 关于专利法第26条第4款

专利法第26条第4款规定，权利要求书应当以说明书为依据，说明要求专利保护的范围。

如果基于说明书充分公开的内容，本领域技术人员能够认定权利要求概括的所有技术方案均能解决发明要解决的技术方案，并达到预期的技术效果，则权利要求书得到了说明书的支持。

请求人认为，本专利说明书仅验证了实施例1制备的产品具有较好溶解度，但权利要求1-3的技术方案均不能直接对应于实施例1，因此不能认为能够解决上述技术问题，得到相应技术效果。

合议组认为，如前文所述，权利要求1-3涉及的产品均为晶型II，本专利说明书已经对该晶型进行了较为详细的制备、测定和表征，符合本领域对晶型的常规表征的一般要求。在没有提供相应证据的情况下，请求人的上述质疑缺乏合理依据，在口头审理中进一步提出的观点，即“必须每个实施例的晶型都需要进行验证，即使在误差范围内也不能认为具有相同的谱图”超出了本领域的基本认知，权利要求1-3不符合专利法第26条第4款规定的无效宣告理由不能成立。

综上所述，请求人主张的本专利权利要求1不符合专利法第22条第2款，权利要求2-3不符合专利法第22条第3款，权利要求1-3不符合专利法第26条第4款规定的无效理由均不能成立，合议组作出如下决定。

## 三、决定

在专利权人于2024年06月03日提交的权利要求1-10的基础上，维持201510038935.3号发明专利权有效。

当事人对本决定不服的，根据专利法第46条第2款的规定，可以自收到本决定之日起三个月内向北京知识产权法院起诉。根据该款的规定，一方当事人起诉后，另一方当事人作为第三人参加诉讼。
