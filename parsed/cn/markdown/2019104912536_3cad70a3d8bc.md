## 决定要点：

1. 是否违反诚实信用原则，需要从以下两个构成要件进行分析：一是客观上专利权人是否存在弄虚作假的行为，二是主观上专利权人是否存在弄虚作假的故意。以不符合专利法实施细则第11条为由提出无效宣告请求的，请求人应当承担充分举证的义务。如果请求人仅是质疑数据不合常理但其举证没有达到证明专利权人故意编造数据、申请专利的行为存在弄虚作假时，请求人关于涉案专利违反专利法实施细则第11条的无效理由不成立。

2. 以说明书为依据，是指权利要求应当得到说明书的支持。如果基于说明书记载的内容、本领域技术人员的常识和请求人举证的现有技术状况，以本领域技术人员的视角，尚无合理的理由怀疑权利要求的概括不能解决发明拟解决的技术问题，则请求人关于权利要求得不到说明书支持的无效理由不成立。

3. 涉案专利权利要求与最接近现有技术之间存在区别特征，现有技术中公开了区别特征，且其在现有技术中所起的作用与其在涉案专利中所起的作用相同，是认定现有技术存在引入区别特征的技术启示的必要条件。如果请求人的所有现有技术证据均未教导该区别特征在涉案专利权利要求技术方案中所起的作用，则请求人关于涉案专利权利要求不具备创造性的理由不成立。

## 二、决定的理由

## 1. 关于审查文本

本决定以本专利授权公告文本为基础作出。

## 2.关于证据

## 3. 关于专利法实施细则第11条

专利法实施细则（2023年修订）第11条规定，“申请专利应当遵循诚实信用原则。提出各类专利申请应当以真实发明创造活动为基础，不得弄虚作假。”

是否违反诚实信用原则，需要从以下两个构成要件进行分析：一是客观上专利权人是否存在弄虚作假的行为，二是主观上专利权人是否存在弄虚作假的故意。以不符合专利法实施细则第11条为由提出无效宣告请求的，请求人应当承担充分举证的义务。如果请求人仅是质疑数据不合常理但其举证没有达到证明专利权人故意编造数据、申请专利的行为存在弄虚作假时，请求人关于涉案专利违反专利法实施细则第11条的无效理由不成立。

本案中，请求人认为本专利化合物的活性试验数据真实性存疑，主要基于以下理由：①本专利说明书中化合物29与结构相似的其他化合物（如化合物32、7-9、104-105）的活性差异较大，不合常理，尤其是化合物104为专利权人最终上市的贝福替尼（参见证据12）的情况下；②本专利说明书提供的是化合物29二甲磺酸盐的活性数据，实审程序中提交的是化合物29的活性数据（参见证据14），二者完全一致不合常理；③证据6、7、16均研究了化合物29，其中突变型和野生型的选择性仅是45倍左右，远低于本专利的194倍；④第三方按照本专利记载的条件进行化合物29的活性测试（证据15），其选择性结果与本专利记载的194倍存在显著差异。

专利权人认为请求人的以上理由不成立：①专利权人首先合成出化合物29，对其进行表征和活性测试，并将结果如实记载于说明书中，专利权人不存在编造数据和技术效果的主观故意或客观行为，专利权人提供的原始实验记录（反证20）也可表明在本专利优先权日前，专利权人已经完成了化合物29的LoVo、NCI-H1975和NCI-2073的细胞活性测试实验；②证据6、7、15和16至多显示不同主体对相同化合物测试的活性数据不完全一致，这是本领域客观存在的普遍现象，这些证据之间也是不一致的；③证据6、7和16记载的生物活性测试方法和条件与本专利不同，例如其中野生型细胞株为A431，而本专利为NCI-2073，之间的活性数据不具有可比性；④证据15、证据15-1中化合物28为奥希替尼（以下简称AZD9291），其活性试验数据显示AZD9291及其甲磺酸盐不具有NCI-2073/NCI-H1975选择性，但AZD9291原研公司阿斯利康的文献报道中，AZD9291具有较高的NCI-2073/NCI-H1975选择性（参见反证21），可见证据15-1本身的数据准确性存疑，证据15和15-1不应当被采信。

## 3.3 合议组的评述意见

双方的争议集中在，本专利说明书中关于化合物29的活性数据是否存在编造和弄虚作假的情况。合议组认为，基于请求人的举证尚无法确认以上事实存在，即请求人的以上理由不成立。

首先，结构类似的化合物具有类似的活性，但是对于构效关系敏感的位置而言，即使微小的基团变化，也有可能引发活性存在较大差异。请求人并未充分举证本专利化合物29与化合物32、7-9、104-105中存在结构差异的部分不属于构效关系敏感的部位，因此基于这些化合物之间活性存在较大差异即推测化合物29的活性数据不真实缺乏事实依据。即使证据12中将化合物104-105另行要求专利保护，也不因此意味着说明书中化合物29的活性数据就存在不真实的情况。

其次，证据14的答复意见虽然在表格中显示的是“本专利化合物29”，但其文字部分明确记载测试的是化合物29的二甲磺酸盐，这与本专利说明书中内容并不矛盾。

再次，本专利与证据6、7、16为不同主体针对相同化合物的研究，虽然其中双突变细胞株均为NCI-H1975，但本专利中的野生型细胞株为NCI-2073（人非小细胞肺癌细胞），证据6、7、16中的野生型细胞株为A431（人上皮细胞癌细胞），测试条件不同的情况下，测试结果不具有可比性。另次，证据15（包括证据15-1）系第三方进行的平行实验，其结果显示化合物29的二甲磺酸盐对于NCI-H2073/NCI-H1975的选择性为 $186.88 / 146.18 = 1.28$ ，化合物29的甲磺酸盐对于NCI-H2073/NCI-H1975的选择性为 $158.28 / 147.01 = 1.08$ ，化合物29的游离碱对于NCI-H2073/NCI-H1975的选择性为 $166.93 / 180.82 = 0.92$ ，即化合物29不具有NCI-H2073/NCI-H1975的选择性，这虽然与本专利说明书记载的数据存在较大差异，但同时合议组也注意到，证据15的数据与请求人提交的多份其他证据中的数据也存在较大差异，具体体现在：①证据15显示AZD9291对于NCI-H2073/NCI-H1975的选择性为 $57.52 / 50.66 = 1.13$ ，AZD9291甲磺酸盐对于NCI-H2073/NCI-H1975的选择性为 $50.63 / 69.79 = 0.72$ ，即AZD9291不具有NCI-H2073/NCI-H1975的选择性，这与反证21，即AZD9291原研药公司阿斯利康的研究存在较大差异；②证据15显示化合物29的二甲磺酸盐对于A431/NCI-H1975的选择性为 $1702.39 / 146.18 = 11.65$ ，化合物29的甲磺酸盐对于A431/NCI-H1975的选择性为 $501.89 / 147.01 = 3.41$ ，化合物29的游离碱对于A431/NCI-H1975的选择性为 $1241.28 / 180.82 = 6.86$ ，这与证据7记载的化合物29对于A431/NCI-H1975的选择性 $105 / 2.3 = 45.6$ 、证据16记载的化合物29对于A431/NCI-H1975的选择性 $>1000 / 22\approx 45.5$ 数据也存在较大差异。在证据15的数据结果客观上与包括本专利在内的多个不同主体给出的数据结果均存在较大差异，且请求人不能对此予以合理解释的情况下，证据15（包括证据15-1）不能证明本专利说明书中记载的活性数据存在编造等情况。

综上，请求人关于本专利化合物的活性试验数据真实性存疑，本专利不符合专利法实施细则第11条规定的无效理由不成立。鉴于此，请求人在其他法条的评述中以这一事实主张为依据的理由，合议组不再重复评述。

## 4.关于专利法第26条第3款

专利法（2008年修订）第26条第3款规定，“说明书应当对发明或者实用新型作出清楚、完整的说明，以

所属技术领域的技术人员能够实现为准；必要的时候，应当有附图。”

能够实现，是指本领域技术人员根据说明书记载的内容就能够实现其技术方案，解决其技术问题并产生预期的技术效果。对于化学产品而言，说明书应当记载化学产品的确认、制备和用途。

本案中，请求人认为本专利权利要求1-5的技术方案公开不充分的理由仅为，本专利说明书关于化合物29的活性和选择性试验数据存疑。如前所分析，在以上理由不成立的情况下，请求人关于权利要求1-5不符合专利法第26条第3款规定的无效理由不成立。

## 5.关于专利法第26条第4款（权利要求应当得到说明书支持）

专利法（2008年修订）第26条第4款规定，“权利要求书应当以说明书为依据，清楚、简要地限定要求专利保护的范围。”

以说明书为依据，是指权利要求应当得到说明书的支持。如果基于说明书记载的内容、本领域技术人员的常识和请求人举证的现有技术状况，以本领域技术人员的视角，尚无合理的理由怀疑权利要求的概括不能解决发明拟解决的技术问题，则请求人关于权利要求得不到说明书支持的无效理由不成立。

本案中，请求人认为权利要求1-5得不到说明书支持的理由主要基于两点：一是权利要求1-5的技术方案未充分公开，二是权利要求1中“药学上可接受的盐”范围过宽，概括了本领域技术人员无法得到的化合物29的碱性盐或者成盐个数为2之外的其他酸加成盐的情形（参见证据8、9）。

首先，请求人关于权利要求1-5的技术方案未充分公开的理由不成立，其也不会影响到权利要求1-5不符合专利法第26条第4款的规定。

其次，根据说明书背景技术及第0026段的记载，本专利的目的在于提供一种化合物，该化合物可以对表皮生长因子受体(EGFR)蛋白激酶的变异形态产生抑制作用，并可以克服现有药物吉非替尼、厄洛替尼等诱发的耐药性。一方面，化合物能否成盐以及能够形成哪些种类的盐是由其化学结构决定的，成盐通常不会实质性改变化合物本身的活性；另一方面，本专利已经验证了化合物29二甲磺酸盐对于双突变型细胞和野生型细胞的抑制活性，且具有NCI-H2073/NCI-H1975的选择性，同时请求人并未提供证据表明哪种“药学上可接受的盐”会影响化合物29的活性。

再者，证据8是本专利申请日后的证据，不能证明申请日前本领域技术人员的认知和能力。证据9第413页第二段公开“碱性药物的成盐”，其作为一般性理论，恰恰教导了本领域技术人员可以根据化合物的结构，知晓如何形成“药学上可接受的盐”。

综上所述，请求人关于权利要求1中“药学上可接受的盐”范围过宽，导致权利要求1-5不符合专利法第26条第4款（支持）的理由不成立。

## 7.关于专利法第22条第3款

专利法（2008年修订）第22条第3款规定，创造性，是指与现有技术相比，该发明具有突出的实质性特点和显著的进步，该实用新型具有实质性特点和进步。

涉案专利权利要求与最接近现有技术之间存在区别特征，现有技术中公开了区别特征，且其在现有技术中所起的作用与其在涉案专利中所起的作用相同，是认定现有技术存在引入区别特征的技术启示的必要条件。如果请求人的所有现有技术均未教导该区别特征在涉案专利权利要求技术方案中所起的作用，则请求人关于涉案专利权利要求不具备创造性的理由不成立。

## 7.1.3.7 合议组的意见

综合双方的意见和请求人评价技术启示的在案证据，合议组认为请求人的意见不成立。具体理由如下：

首先，请求人所有证据结合方式的前提是，本专利说明书中有关NCI-H2073/NCI-H1975选择性的数据不能被采信，但如前第3节所分析，这一事实主张并不成立。在此情况下，判断现有技术中是否存在技术启示，就需要以7.1.2节所确定的发明实际解决的技术问题（即提高化合物的NCI-H2073/NCI-H1975选择性）为目标，考察来自现有技术的技术信息是否能够对解决以上技术问题有所启发。事实上，以上任何一份现有技术文献中均未涉及化合物对于NCI-H2073野生型细胞系的抑制作用。

其次，现有技术中公开了区别特征，且其在现有技术中所起的作用与其在涉案专利中所起的作用相同，是认定现有技术存在引入区别特征的技术启示的必要条件。事实上，以上任何一份现有技术文献中均未公开相同或类似结构化合物苯环上的连接基为-O-时，所述基团如何影响化合物对于野生型/突变型细胞的选择性（尤其是NCI-H2073/NCI-H1975选择性），即未教导与本专利中起相同作用的基团替换，在此情况下，不能认定所述现有技术文献给出了引入区别特征的技术启示。

再者，现有技术中的教导应当是明确、具体的，使得本领域技术人员在解决发明实际解决的技术问题，有动机针对性地引入所述区别特征来解决该技术问题，是认定现有技术存在引入区别特征的技术启示的基本要求。本案中，证据1中与区别特征相应的取代基为 $R^{3}$ ，示例的所有基团均为通过-N-连接到苯环上的取代基，虽然吉非替尼、厄洛替尼的相应位置为-O-，但是在看到本发明之前，本领域技术人员不会意识到-O-和-N-对于化合物选择性的作用，从而有动机将AZD9291中的“-N-甲基”替换为“-O-”来解决化合物针对NCI-H2073/NCI-H1975选择性的问题。证据19、3和2的研究是接续进行的，最终筛选得到AZD9291，其中亦未就苯环上的连接基是如何影响化合物针对野生型/突变型的选择性（特别是NCI-H2073/NCI-H1975选择性）给出明确具体的指引。证据10、11和13中，不仅化合物的母核结构与本专利存在诸多差异，而且同样未关注到相应位置的连接基对于野生型/突变型的选择性的影响。证据5仅是关于电子等排体的一般性理论，其提供的仅是药物设计时的一种参考，并未对本案特定结构的化合物解决特定的技术问题给出明确、具体的教导。

综上，请求人关于本专利权利要求1不具备创造性，不符合专利法第22条第3款的规定的理由均不成立。

## 7.1.4 关于技术效果

请求人在请求书的第67-77页，还进一步对权利要求1未产生预料不到的技术效果单独进行论述。根据《专利审查指南》第二部分第四章第5节的规定，发明是否具备创造性，通常应当根据“三步法”进行审查。一方面，如果根据“三步法”已经得出发明非显而易见的结论，则通常勿需进一步考虑该节规定的“其他因素”；另一方面，发明是否取得了预料不到的技术效果，在“三步法”确定发明实际解决的技术问题和判断是否存在技术启示时通常会进行实质性考虑。本案即属于以上情况，因此，对于请求人的相应理由，合议组不再作重复评述。

以下仅就前文未涉及的证据4评述如下。请求人使用证据4，用于证明本专利化合物29二甲磺酸盐对LoVo/NCI-1975的选择性（193.77倍）相比证据1中AZD9291甲磺酸盐对LoVo/NCI-1975的选择性（113.51倍），未产生任何预料不到的技术效果。

经查，证据4公开如下内容，“在结合测定中有几个问题可以导致错误的实验结果。在放射受体结合测定中，最常见的问题是放射配体的分解。它可表现为特异性结合的减少和/或非特异性结合的增加。不过，这种情况只是偶尔出现。绝不能用已过失效期的不稳定的放射性配体。另一个问题是受体在保存或孵育过程中的分解。因此，组织应避免过久的保存或反复的冻结和解冻。为了监测结合分析的可靠性，我们在每次实验中均作一项阳性对照物，比较本次实验中它的 $\mathrm{IC}_{50}$ 和以往的 $\mathrm{IC}_{50}$ 值，如果相差在3倍以上，则应对本次实验有所怀疑。”

合议组认为，一方面，根据《专利审查指南》第二部分第四章3.2.1.1节的规定，“发明的任何技术效果都可以作为重新确定技术问题的基础，只要本领域技术人员从该申请说明书中所记载的内容能够得知该技术效果即可。”以上在确定发明实际解决的技术问题时，依据的是NCI-2073/NCI-1975的选择性，并非LoVo/NCI-1975的选择性。另一方面，证据4的以上内容仅表明当两次实验“相差3倍以上”时，需要考虑实验的准确性，并不意味着针对不同的细胞系相差较大，数据就一定不准确。因此，请求人的以上意见不成立。
